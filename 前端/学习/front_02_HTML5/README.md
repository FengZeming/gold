| 属性   |值   | 描述   |
| ------------- |
| manifest     |URL|定义一个 URL，在这个 URL 上描述了文档的缓存信息。|
| xmlns     |	http://www.w3.org/1999/xhtml|HTML 不支持。只有 XHTML 支持。|

2.文本格式化标签

HTML 5 规范声明：
- 标题应该用 <h1> - <h6> 标签表示。
- 被强调的文本应该用 <em> 标签表示。
- 重要的文本应该用 <strong> 标签表示。
- 被标记的或者高亮显示的文本应该用 <mark> 标签表示。
